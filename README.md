# Online Banking System

Full-stack online banking application built using Angular and Spring Boot.

## Tech Stack
- Angular
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL

## Features
- User Registration & Login
- JWT-based Authentication
- Account Dashboard
- Fund Transfer
- Transaction History

## How to Run

### Backend
- Create MySQL database `online_banking`
- Configure `application.properties`
- Run Spring Boot app

### Frontend
```bash
npm install
npx ng serve
